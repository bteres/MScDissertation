\abstract{\addtocontents{toc}{\vspace{1em}} % Add a gap in the Contents, for aesthetics

The work presented contributes to research in lightning protection simulations and focuses on approximating a lightning waveform that can be transformed into the frequency domain. Previous work in this area has produced very specific forms of the Heidler function (lightning current waveform). This work however focuses on a generic solution with parameters that can be modified to produce any lightning waveform that is required. In the research presented, the ``Terespolsky function'' is obtained. This function has an analytical solution in the frequency domain and has constants that can be changed to obtain different waveforms (8/20, 10/350, etc.). The characteristics of the ``Terespolsky function'' are compared with those of the Heidler function to ascertain whether or not the function is applicable for use with the lightning protection standard (IEC 62305). It is shown that the ``Terespolsky function'' does represent the same characteristics as those of the Heidler function and hence the IEC 62305. A practical application of the function in the frequency domain is carried out to synthesise a circuit network that generates a lighting impulse current (the ``Terespolsky function'') through a load. This represents a valuable contribution to engineers working in the field of lightning protection, specifically simulation models.
}
